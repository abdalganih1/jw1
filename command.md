## تعليمة جاهزة لإرسالها إلى AI Agent (انسخها كما هي)

**المهمة:**
أنت مصمّم UI/UX ومطوّر واجهات Front-End خبير. صمّم لي واجهات UI أمامية لموقع لعرض وبيع وتصميم المجوهرات. المطلوب واجهات فاخرة (Luxury)، نظيفة، حديثة، ومتجاوبة بالكامل (Responsive) للويب والموبايل.

### 1) هدف الموقع

* عرض منتجات مجوهرات (خواتم، أساور، قلائد، أقراط) بصور **حقيقية** عالية الجودة.
* توفير تجربة “تصميم مجوهرات” (Custom Jewelry Builder) تسمح للعميل باختيار المعدن/الحجر/القياس/الحفر… مع معاينة مباشرة.

### 2) متطلبات التصميم البصري (Visual Style)

* طابع فاخر: خلفيات فاتحة/داكنة أنيقة، مساحات بيضاء مريحة، خطوط راقية.
* استخدام Micro-interactions: Hover، انتقالات ناعمة، Skeleton loading.
* هوية ألوان مقترحة: (ذهبي/عاجي/أسود فحمي) مع بدائل.
* صور المنتجات يجب أن تكون **حقيقية** (ليست رسومات كرتونية).

  * استخدم صورًا من مصادر صور مجانية موثوقة مثل Unsplash/Pexels/Shopify demo images، أو اربطها كمراجع قابلة للاستبدال لاحقًا.
  * ممنوع استخدام صور منخفضة الجودة أو AI-looking.

### 3) الصفحات/الشاشات المطلوبة (Deliverables)

صمّم على الأقل الشاشات التالية مع نسخة Desktop وMobile:

1. **الصفحة الرئيسية (Home):**

   * Hero فاخر + CTA (تسوّق الآن / صمّم قطعتك)
   * أقسام: الأكثر مبيعًا، مجموعات جديدة، قصتنا، تقييمات، Instagram-like gallery
2. **صفحة المتجر/الكتالوج (Shop/Collection):**

   * فلاتر متقدمة: النوع، المعدن، الحجر، السعر، المناسبة، التوفر
   * فرز: الأكثر شيوعًا/السعر/الجديد
   * Grid بطاقات منتج احترافية
3. **صفحة تفاصيل المنتج (PDP):**

   * معرض صور متعدد + Zoom
   * خيارات: قياس، معدن، حجر، نقش، تغليف هدية
   * سعر ديناميكي + “أضف للسلة” + “احجز استشارة”
4. **صفحة تصميم المجوهرات (Jewelry Builder):**

   * خطوات واضحة (Stepper): اختر النوع → اختر المعدن → اختر الحجر → المقاس → الإضافات → المعاينة
   * معاينة مباشرة (على الأقل صور تتبدل حسب الاختيارات)
   * ملخص السعر + مدة التسليم المتوقعة
5. **السلة والدفع (Cart/Checkout):**

   * سلة أنيقة + كوبونات + شحن + ملخص
   * Checkout بسيط خطوة بخطوة
6. **حساب المستخدم (Profile/Orders):**

   * طلباتي، العناوين، المفضلة، تتبع الطلب
7. **صفحات مساعدة:** FAQ، سياسة الإرجاع، تواصل معنا، صفحة حجز موعد/استشارة

### 4) نظام المكوّنات (UI Components)

أنشئ مكتبة مكوّنات متسقة تشمل:

* Navbar ثابتة + Mega menu (اختياري)
* Product Card (صور حقيقية، شارات: New/Best Seller)
* Filter Sidebar/Top filters
* Image Gallery + Zoom
* Stepper/Progress للـ Builder
* Modal للحفر/اختيار الحجر
* Toast/Notifications
* Skeleton loaders
* Footer غني بروابط وثقة (ضمان/شحن/دفع)

### 5) المحتوى والنسخ (Copy)

* اكتب نصوص عربية فاخرة مختصرة للعناوين والأزرار ووصف الأقسام (مع إمكانية إضافة English لاحقًا).
* اجعل الـ Tone: راقٍ، بسيط، غير مبالغ.

### 6) الوصولية والأداء (A11y & Performance)

* تباين ألوان مناسب، دعم لوحة المفاتيح، Alt للصور.
* تحسين صور (Lazy loading) وتقليل CLS.

### 7) المخرجات المطلوبة منك بالضبط

أخرج لي التالي بشكل منظم:

1. **خريطة صفحات (Sitemap)** + تدفق المستخدم (User Flow).
2. **Wireframes مختصرة** لكل صفحة (نقاط/مخططات).
3. **تصميم UI نهائي** موصوف بدقة (ألوان/خطوط/Spacing/مقاسات).
4. **تصميم متجاوب** مع نقاط توقف (Breakpoints).
5. إن كنت تكتب كود:

   * نفّذ الواجهات بـ **React + Tailwind** (أو Next.js)
   * اجعل الصور كمراجع URLs جاهزة للاستبدال
   * ضع بيانات منتجات Mock واقعية (اسم، سعر، معدن، حجر، وزن، صور متعددة)
6. سلّم الملفات/المكوّنات بطريقة منظمة وقابلة للتشغيل.

### 8) قيود مهمة

* لا تستخدم تصاميم عامة مكررة؛ أريد لمسة إبداعية مع المحافظة على الفخامة.
* الصور يجب أن تكون واقعية وعالية الجودة.
* حافظ على تجربة مستخدم واضحة وسهلة (خصوصًا في صفحة التصميم Builder).

**ابدأ الآن بتقديم 3 اتجاهات تصميم (Design Directions) مختلفة** مع وصف مختصر لكل اتجاه، ثم اختر أفضل اتجاه وطبّقه على كل الشاشات.
